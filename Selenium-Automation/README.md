
# Selenium Automation – Demo Web Shop

## Project Overview
This project contains Selenium WebDriver automation scripts for testing the Demo Web Shop e-commerce website.

## Application Under Test
https://demowebshop.tricentis.com

## Tools Used
- Java
- Selenium WebDriver
- Eclipse IDE
- Chrome Browser

## Automated Test Scenarios
1. User Registration
2. User Login
3. Product Search
4. Add Product to Cart
5. Logout
6. First Name validation with special characters
7. Last Name validation with special characters

## Bug Validation
During testing, the following validation issues were observed:
- First Name field accepts special characters.
- Last Name field accepts special characters.
- Weak password such as `nnnnnn` is accepted.

## How to Run
1. Open project in Eclipse.
2. Add Selenium JAR files to Build Path.
3. Open any test class.
4. Right-click → Run As → Java Application.

## Test Website
Demo Web Shop is used for practice automation testing.
